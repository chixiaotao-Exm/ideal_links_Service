# data/

运行时数据目录。

部署后程序会在这里保存本地数据库或运行中间文件，例如 CDK 数据。公开仓库只保留这个说明文件。

不要提交：

- `*.sqlite3`
- `*.sqlite3-*`
- `confirm_body.txt`
- `paypal_confirm_post.txt`
- 诊断输出
- 真实请求/响应数据
- Token、代理、Cookie、订单号、支付长链
